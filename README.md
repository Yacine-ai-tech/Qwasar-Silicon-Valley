## üöÄ Getting Started

### Prerequisites
- **Python 3.x** for Python projects
- **GCC** or **Clang** compiler for C projects
- **Make** for building C projects
- **pip** for installing Python dependencies

### Running Python Projects
```bash
cd "Preseason Data/<project-name>/<project-name>"
python <main-file>.py
```

### Running C Projects
```bash
cd "Season 1 Arc 1/<project-name>/<project-name>"
make
./<executable-name>
```

### Running Data Science Projects
```bash
cd "Season 2 Data Science/<project-name>/<project-name>"
pip install -r requirements.txt  # if requirements.txt exists
python <main-file>.py
```
